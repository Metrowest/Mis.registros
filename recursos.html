Instalación para Windows

Este Web App le permite instalarlo en una computadora, con sistema operativo Windows, como una aplicación. Para instalar la aplicación deberá abrir el enlace en un navegador como Microsoft Edge o internet Explorer

Luego desde la pagina donde está la aplicación diríjase a los 3 puntos en la parte superior derecha

A continuación diríjase a la opcion “App” y presione “instal this site as an app"

Presione intall

Seleccione su preferencia (se recomienda las que están marcadas en la foto para tener acceso directo en su escritorio y en la barra de tareas al pie de la página

Resultado si selecciona crear acceso en la barra de tareas (Pin to taskbar)


Instalación para Android
